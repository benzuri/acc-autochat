---------
- Start:
Start the program to begin detecting if you have pressed a shortcut, in that case ACC Autochat will write the associated text in current application.
See option 'Display messages only in ACC' to display messages only if you are in ACC.
---------
- Stop:
Shortcut detection ends.
---------
 -Shortcuts:
Edit the shortcut configuration file.
The first field is the key to use and the second the text to write in the chat.
---------
 -Display messages only in ACC:
If this option is activated (and the application started), messages will only be displayed in ACC and not in another application.
